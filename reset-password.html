📋 Cara Reset Password:

  1. Masukkan email Anda di form atas
  2. Klik "Kirim Link Reset"
  3. Buka email inbox Anda (cek spam juga)
  4. Klik link reset password di email
  5. Set password baru di halaman yang muncul
  6. Login dengan password baru
